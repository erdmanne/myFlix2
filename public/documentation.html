Documentation myFlix App

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies. Structure:

{ id: 1, title: "King Lion", genre: "family", description: "good to watch with your family", director: "Walt Disney" },

Get data about a single movie by title /movies/[title] GET None JSON object holding data about a single movie. Structure same as above.
Get movies by genre /genre/[genre] GET None JSON object holding data about a movie genre, Structure:

{ id: 1, title: "King Lion", genre: "family", description: "good to watch with your family", director: "Walt Disney" },

Add a movie to the user's favorites list /favorites/[username]/movies/[title] POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favorites/[username]/movies/[id] DELETE None A message indicating the movie title was deleted from the user's favorites list.
